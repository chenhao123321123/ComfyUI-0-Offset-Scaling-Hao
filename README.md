# ComfyUI-0-Offset-Scaling-Hao
一套专为图片扩展场景设计的配套节点包。解决大部分编辑模型带来的偏移问题

## 包含节点
1. 图片智能缩放：等比例缩放 + 64对齐 + 居中，输出处理后图片、绘制遮罩、扩展遮罩、原图尺寸字符串（拼接）
2. 还原尺寸：根据扩展遮罩裁剪图片，再按「拼接」字符串还原为原图尺寸

## 使用方法
整体连线逻辑适用于**图片编辑**与**局部重绘**场景，二者连接方式基本一致：
1. 基础连接
   - 原图 + 遮罩 → 接入 **图片智能缩放**
   - **还原尺寸** 的 image 端，接入模型生成后解码出来的图像
   - **还原尺寸** 的「扩展遮罩」「拼接」，分别接入图片智能缩放的对应输出接口
2. 场景区分
   - 局部重绘：完整连线，需将「绘制遮罩」接入重绘相关节点
   - 图片编辑：无需连接「绘制遮罩」，其余接口正常连接即可
