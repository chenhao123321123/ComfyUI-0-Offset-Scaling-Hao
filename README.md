# ComfyUI-0-Offset-Scaling-Hao

一套专为图片扩展场景设计的配套节点包。解决大部分图片编辑模型像素偏移问题，只有两个节点

## 包含节点
1.  **图片智能缩放**
    - 功能：等比例缩放 + 64对齐 + 居中
    - 输出：处理后图片、绘制遮罩、扩展遮罩、原图尺寸字符串（拼接）

2.  **还原尺寸**
    - 功能：根据扩展遮罩裁剪图片，再按「拼接」字符串还原为原图尺寸
    - 输入：处理后图片、扩展遮罩、拼接字符串
## 使用方法
整体连线逻辑适用于**图片编辑**与**局部重绘**场景，二者连线基本一致，仅一处区别：
1. 基础连线
   - 原图 + 自定义遮罩 → 接入 **图片智能缩放**
   - 图片智能缩放的「扩展遮罩」「拼接」，分别对应接入 **还原尺寸** 的同名输入端
2. 场景区分
   - 局部重绘：完整连线，需将「绘制遮罩」接入重绘相关节点
   - 图片编辑：无需连接「绘制遮罩」，其余接口正常连线即可
